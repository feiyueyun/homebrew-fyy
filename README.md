# Homebrew Tap for FYY CLI

Official [Homebrew](https://brew.sh) tap for [FYY CLI](https://github.com/feiyueyun/fyy).

## Installation

```bash
brew install feiyueyun/tap/fyy
```

## Usage

```bash
# Check version
fyy version

# Join a mesh network
fyy network join --authkey=tskey-auth-xxx

# Check status
fyy status
```
